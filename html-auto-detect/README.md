# HTML Auto Detect & Preview

VS Codeエディタ内のHTMLコードを自動検知して、ファイルとして保存する拡張機能。

## 機能

- エディタ内容の変更を監視
- 完全なHTMLコード（`<!DOCTYPE html>`〜`</html>`）を自動検知
- 3秒間変更がなければ自動的にファイルとして保存
- エディタからHTMLコードを自動削除（切り取り動作）
- ステータスバーで有効/無効切り替え

## 使い方

1. watch.jsを起動
2. VS Codeでこの拡張機能を有効にする
3. AIがHTMLコードを出力 → 自動的に保存＆ブラウザ表示
